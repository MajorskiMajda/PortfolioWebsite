SecretsProject is a web application that lets users securely share secrets anonymously. It is built using Node.js and Express.js for the back end, with EJS (Embedded JavaScript) templates powering the dynamic front end. The project incorporates MongoDB for data storage, hosted on MongoDB Atlas. The app demonstrates user authentication using OAuth and passport.js, ensuring secure login with third-party providers. This project highlights skills in secure full-stack application development, authentication integration, and database management.
