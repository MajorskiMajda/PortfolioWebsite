Azure is a full-stack e-commerce project showcasing a dynamic shopping experience. The front end is built in React, provideing a modern, responsive UI with state management. The back end, developed using Node.js and Express, handles server-side logic and RESTful API communication. The app supports modular development, enabling seamless integration between client and server. The front end is hosted on GitHub Pages, the back end is hosted on Render. Data is stored in a MongoDB database, hosted on MongoDB Atlas.
