KeeperApp is a full-stack web application inspired by Google Keep. The front end is developed using React, offering a clean and interactive interface for creating, viewing, and deleting notes. The back end is powered by Node.js and Express, handling server-side functionality and database interactions. Notes are stored securely in a MongoDB database hosted on MongoDB Atlas. This project showcases skills in React, RESTful APIs, and full-stack application development while emphasizing simplicity and usability.
